# Copywriting Guide: WhatsApp School Communication

## ЁЯОп Voice & Tone Guidelines

### Brand Voice: **Trustworthy Educational Partner**
- **Professional yet Warm**: Official but approachable
- **Respectful**: Honors parents and cultural values
- **Supportive**: Encouraging, never judgmental
- **Clear**: Simple language, no educational jargon
- **Consistent**: Predictable format and style

## ЁЯУЭ Writing Principles

### 1. **Clarity First**
- Use simple, everyday words
- Avoid educational jargon
- One main idea per message
- Clear action items (if any)

### 2. **Respect Cultural Context**
- Always start with "ЁЯЩП Namaste"
- Use appropriate honorifics
- Acknowledge family values
- Respect religious considerations

### 3. **Emotional Intelligence**
- Lead with empathy
- Acknowledge parent concerns
- Celebrate small wins
- Offer support, not criticism

### 4. **Practical Structure**
- Start with context (what/who)
- Provide key information
- End with encouragement
- Include contact for questions
